Technical Notes

  • Image to Text: Uses Tesseract.js for client-side OCR
  • PDF to Word: Uses PDF.js for text extraction and docx.js for Word generation
  • PPTX to PDF: Uses jsPDF to create instruction PDFs (full conversion requires server-side processing)
  • Dependencies: All required libraries are loaded via CDN
  • Browser Support: Modern browsers with ES6+ support
